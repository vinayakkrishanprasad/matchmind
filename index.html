Methodology

Open source · GitHub
01
Poisson Goal Model
Each team's expected goals are estimated using a Dixon-Coles Poisson regression on 4 years of international data. Attack and defense parameters update after each finished match. Scoreline distributions are simulated to produce win/draw/loss probabilities.
02
Media Sentiment
Headlines and descriptions from NewsAPI and GNews are scored using a 300+ term football-specific weighted lexicon — multi-word phrases like "red card" and "clean sheet" are matched before single words to avoid double-counting.
03
Momentum Score
Calculated directly from match results: recency-weighted win/draw/loss across the last 3 games, average goal difference, and a clean sheet bonus. This is the model's read on actual recent form, separate from media narrative.
04
Market Calibration
After each match, probabilities are compared against closing odds. Brier scores are computed for both the sentiment-adjusted model and the ELO baseline, giving a running measure of whether the sentiment layer adds real predictive value across all 104 matches.
